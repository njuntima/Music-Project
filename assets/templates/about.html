About MusicDB

This application is meant to be a Spotify clone. It provides users with the ability to create playlists, upload songs, and enjoy a music streaming experience.

User Types

Admin

The Admin can create/remove users, edit song data

Artist

The Artist can create albums and songs

User

The User can create playlists and add/remove songs in their playlists
